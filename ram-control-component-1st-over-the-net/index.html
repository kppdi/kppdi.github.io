RAM Control Component - 1st Over The Net!

Grup KPPDI 20 September 2007

Saat ini sudah banyak aplikasi yang menawarkan kemampuan menampilkan informasi penggunaan memory, dalam bentuk label, grafik, histogram, dsb. Bahkan ada juga yang menyediakan fasilitas defragmen/Cleanup RAM dari memory leak, lost clipboard information atau data dari aplikasi yang berakhir prematur. Sebut saja Tweak-XP, RAM Optimizer, RAM XP, dsb.

Tapi, ada tapinya nih, sejauh yang saya googling, belum ada komponen yang memiliki kemampuan seperti di atas yang freeware, yang bisa didownload beberapa KB, installnya mudah, trus tinggal drop ke form dan tekan F9, semua fasilitas di atas sudah kita punya!

Dua hari lalu, saat begadang menanti sahur, iseng-iseng bikin TRAMCtrl, berbasis pada source code yang dipake di FolderCustomizer.

Fitur yang menonjol dari TRAmCtrl mungkin adalah kemampuannya untuk membersihkan RAM dari memory-leak dan teman-temannya, seperti yang disebut di atas.

Download
Silahkan download TRAMCtl:
TRAMCtrl 3KB
TRAMCtrl Demo 521KB

Catatan: Sudah download ganti ekstensi dari .ppt ke .zip

Instalasi
Ekstrak file TRAMCtrl.zip (yang tadinya TRAMCtrl.ppt, kan sudah direname) ke folder c:\Tramctrl atau lainnya sesuai yang Anda inginkan. Jalankan Delphi (TRamCtrl dites dan berjalan tanpa masalah pada Delphi3, Delphi6 & Delphi7), Klik menu Component>Install Component> Into Existng Package. Pilih file URAM.pas dari folder hasil ektraksi. Klik install. Tambahkan path folder hasil ekstraksi ke Library paths (dari menu Tools>Environment Options>Library).

Penggunaan
Contoh penggunaan tersedia dalam file TRamCtrlDemo.zip hasil download. Terdapat tiga buah demo project untuk tiga versi Delphi: Delphi3, Delphi6 dan Delphi7.
