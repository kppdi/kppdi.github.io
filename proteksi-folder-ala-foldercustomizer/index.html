Proteksi Folder Ala FolderCustomizer

Grup KPPDI

FolderCustomizer memiliki fitur penguncian folder. Idenya sangat sederhana, yaitu menambahkan CLSID shell folder ke akhir nama folder, diawali dengan dot (titik).

Misalnya saya mempunyai sebuah folder, c:New Folder, sbb:

Kemudian saya ganti namanya dengan MoveFile ('c:\New Folder', 'c:\New Folder.{20D04FE0-3AEA-1069-A2D8-08002B30309D}'); //tanpa spasi.

Yang terjadi? Sim salah ketik:

New folder telah berubah menjadi icon my computer. Jika dibuka, isinya adalah drive c:, d:, Control Panel, dsb.

Semoga ada manfaatnya.
