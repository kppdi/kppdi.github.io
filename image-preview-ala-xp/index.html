Image Preview Ala XP

Grup KPPDI

Teraplikasi pada: Delphi3 ke atas, Windows XP ke atas.

Di Delphi, kita semua tidak asing lagi pada yang namanya TImage, yang biasanya dipakai untuk menampilkan gambar. Tetapi kadang-kadang kebutuhan akan tampilan gambar yang fleksibel sulit untuk dipenuhi hanya dengan TImage.

Dalam hal ini saya memiliki ide, menampilkan gambar dengan gaya XP, yang memiliki fungsi rotate, zoom in, zoom out, print, best fit, actual size, dan berbagai fungsi lainnya yang tidak terdapat pada TImage. Ide ini dengan memanfaatkan TPreview dari Windows Preview Library (shimgvw.dll) yang terdapat pada WIndows XP ke atas, di folder System32

Kita lanjut saja ke langkah-langkah instalasi:

  • Jalankan Delphi, tutup semua proyek/file yang terbuka di Delphi.
  • Klik menu Component>Import ActiveX Library
  • Klik Add... dan pilh file C:\Windows\System32\shimgvw.dll. Klik Open.
  • Setelahnya, dari daftar carilah item dengan nama Preview 1.0 Type Library. Pada Class Name otomatis terdapat TPreview.
  • Tentukan Palette Page yang sesuai. Misalkan pilih Standard (defaultnya ActiveX).
    Bandingkan dengan gambar berikut:
  • Klik Install. Pilih Into Existing Package. Klik OK.
  • Ikuti langkah-langkah sisanya sampai selesai dan klik File>Save.

Instalasi telah selesai. Bila sukses, pada palet Standard telah terinstall sebuah komponen baru, TPreview, seperti pada gambar berikut:

Penggunannya cukup mudah:

  • Buat sebuah aplikasi baru, tambhkan TPreview dari page Standard ke form, sesuikan lebar, tinggi dan posisinya pada form.
  • Kemudian tambahkan sebuah button dan OpenPictureDialog.
  • Klik ganda button1, pada event onclick ketik kode berikut:

    [sourcecode language='delphi']

    procedure TForm1.Button1Click(Sender: TObject);

    begin

    if OpenPictureDialog1.execute then

    begin

    Preview1.ShowFile(OpenPictureDialog1.FileName, 1);

    end

    end;

    [/sourcecode]

  • Save dan jalankan aplikasi Anda.

Selamat mencoba
