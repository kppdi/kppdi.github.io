FileDisk Virtual Driver GUI - Updated

Ini updatenya FileDisk Virtual Driver GUI.

Change History:

  • Proteksi password untuk mencegah user selain Anda mengakses drive privat Anda.
  • Menu untuk membersihkan catatan file-file yang pernah dimount sebagai drive, untuk menjaga privasi.
  • Autorun tidak langsung memount drive yang belum diunmount saat komputer dimatikan. Lagi, untuk privasi Anda.
  • Core, kemarin dikompile dengan Visual C++ 2005 (MS. Visual Studio 2005), tapi saya ganti dengan Dev C++ dari BloodShed. MS C++ mengharuskan penyertaan .Net Framework (C++ Runtime Library). Jadi saya pake Dev C++ yang hanya menghasilkan footprint kurang-lebih 19KB saja.
