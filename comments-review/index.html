Beberapa saat lalu, seseorang pengunjung blog ini telah menulis komentar dengan dua pertanyaan:

  1. Bagaimana menambahkan sebuah shortcut untuk meng-uninstall sebuah aplikasi ke Start Menu?
  2. Apa software instalasi selain InstallShield Express bawaan Delphi, yang dapat diuabh tampilan setupnya sesuai kebutuhan?

Well, untuk yang nomor 1., saya mohon maaf, sudah lama ga’ pake InstallShield, jadi lupa di bagian mana shortcut uninstalasi bisa diatur. Jadi saya akan mulai dengan nomor 2.

Yang paling populer dalam membuat paket instalasi software yang dibuat dengan Delphi, saya bisa sebut Inno Setup oleh Jordan Russel. Inno Setup, jadi populer karena gratis & open source, dukungan 3rd party, dan juga dengan hanya beberapa klik pada Wizard-nya, Anda sudah dapat membuat sebuah paket instalasi lengkap dengan shortcut uninstall-nya. Yang lebih cocok sebagai pengikat Inno Setup dengan Delphi adalah penggunaan Pascal Script oleh Carlo Kok, sehingga dalam membuat paket instalasi, Anda bisa menyertakan kode-kode Object Pascal untuk membuat custom action, dialog, memproses file di komputer target, meminta input serial number, dan mengubah tampilan setup sesuka Anda! Namun jika lewat kode masih terasa merepotkan, Inno Setup Form Designer oleh Marc Hoffmann akan membantu Anda mendesain, preview, dan kompile tampilan setup Anda layaknya Anda mendesain tampilan aplikasi di Delphi! What a great setup maker. And they are absolute free!

“Inno Setup is a free installer for Windows programs. First introduced in 1997, Inno Setup today rivals and even surpasses many commercial installers in feature set and stability.”
– Jordan Russel -

Anda bisa saja mencoba Wise, Vise Installer, Setup2Go, Installer2Go, dsb., dengan catatan: BAYAR!

Kembali ke nomor 1., jika Anda menggunakan Inno Setup, buatlah sebuah paket instalasi dengan Wizard, yang kemudian bisa Anda modifikasi. Di sana tersedia opsi untuk menyertakan shortcut uninstall aplikasi Anda. Baca juga dokumentasinya untuk informasi lebih jauh.
