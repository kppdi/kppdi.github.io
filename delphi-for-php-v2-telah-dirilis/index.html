Delphi For PHP v.2 telah dirilis

Grup KPPDI

Setelah merilis Delphi For PHP 1.0, yang menuai kiritik dan saran dari berbagai sumber, kini Borland, di bawah nama CodeGear merilis lagi Delphi For PHP versi 2.

Beberapa fitur baru dan perbaikan telah dibuat untuk versi ini. Antaralain, seperti dikutip dari Delphi For PHP 2.0 what's new:

  • Perbaikan dan tune-up pada IDE, meliputi Visual HTML Designer, Form Designer dengan template, dan dukungan multi-bahasa. Juga, bentuan mendesain, profiling, dan manajemen file.
  • Code Editor telah mendapatkan performansi dan stabilitas: code-folding, error insight, catatan perubahan (change tracking), sync edit dan pemformatan source code untuk mempermudah manajemen dan navigasi terhadap kode-kode.
  • Database juga mendapat perhatian. Pembuatan aplikasi database kini dapat dengan dengan mudah dilakukan, menggunakan lebih sedikit penulisan kode. Penambahan komponen natif untukakses database Oracle, ditambah dukungan tujuh database lainnya, dilengkapi lagi dengan Database Explorer, menjadi pendukung performansi fitur ini.
    Dengan Drag and Drop dari Database Explorer ke form designer, user sudah bisa mengakses database yang didefinisakn untuk digunakan dalam aplikasi. Cepat, dan tanpa kode!
