FileDisk Virtual Disk Drive GUI

Grup KPPDI

Kali ini saya membuat sebuah contoh aplikasi antarmuka dengan command line application dan juga contoh menginstall dan menjalankan Device Driver Service secara on-the-fly. Sekaligus juga contoh penggunaan TTrayIcon pada Delphi (BDS) 2006 for Win32.

Aplikasi ini berguna untuk mounting file ISO (CD Image). Juga berguna untuk membuat dan mounting file-based virtual drive, read-write maupun write only. Berikut tampilan utamanya.

  1. Install Driver
  2. Start Driver
    Jika langkah 1 dan dua gagal, Anda mungkin perlu merestart komputer Anda.
  3. Klik kanan pada daftar yang tersedia, dan pilih opsi-opsi dari popup menu yang tersedia
  4. Pilih Mount, kemudian tentukan parameternya

    Pilih nama file, kemudian tentukan apakah file yang dipilih adalah file CD Image (ISO, BIN, CUE, dll).
    Jika file adalah file-based virtual drive, Anda bisa memilih opsi read-only.
    Kemudian tentukan di drive mana (yang tak terpakai) akan dimount file tersebut sebagai virtual drive.
    Jika file yang dipilih sudah ada sebelumnya, opsi Drive Size akan diabaikan. Jika belum ada, akan dibuat yang baru dengan size yang dipilih.

    Pembuatan file baru tidak dapat dilakukan untuk tipe CD Image.

  5. Sebagai catatan, aplikasi ini hanya berjalan pada Windows 2000 ke atas dan dalam account Administrator/Power User.
