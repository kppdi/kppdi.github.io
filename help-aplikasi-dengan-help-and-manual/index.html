Help Aplikasi Dengan Help & Manual

Grup KPPDI

Help And Manual

[Jika gambar terpotong, klik kanan pada gambar dan open Link In New Window ( FireFox)]

Help And Manual adalah sotware pembuatan file help untuk aplikasi. Help And Manual mendukung beragam format file help, seperi WinHelp (Classic), HTML Help, HTML, dan lain-lainnya. Software ini dapat didownload trialnya di www.helpandmanual.com. Untuk membuat output HTML Help, kita juga perlu software HTML Help Compiler (HHC) dari Microsoft. Tanpa HHC, kita bisa menggunakan WinHelp sebagai alternatif meskipun saat ini HTML Help lebih banyak dipakai, dan WinHelp digunakan hanya untuk menjaga kompatibilitas dengan Windows versi lawas.

Membuat File Help

Selanjutnya kita akan ikuti langkah-langkah pembuatan file help. Karena saya juga tidak memiliki HHC, maka saya menggunakan WinHelp saja dalam contoh ini. Berikut langkah-langkahnya:

  • Jalankan Help & Manual. Buat project baru dari menu File>New. Ikuti saja langkah-langkah yang ada pada wizard sampai selesai.
  • Anda akan mendapatkan sebuah project baru kira-kira seperti gambar berikut ini
  • Pada Welcome Topic, masukkan deskripsi seperlunya pada Page Editor. Bagian ini yang akan ditampilkan nantinya sebagai help (bantuan) bagi user aplikasi kita
  • Setelah selesai, kita pindah lagi ke tab Topic Options

    Di sini, kita hanya perlu memasukkan nilai pada Help Context Number. Pada Delphi, angka ini yang akan menentukan topik mana yang akan ditampilkan pada Context Sensitive Help.
  • Simpan project kita. Sekarang kita lanjutkan dengan proses kompilasi. Pilihlah menu kompilasi seperti gambar di bawah

    Karena di atas tadi kita akan menggunakan WinHelp Classic, maka pilihlah submenu Classic WinHelp pada daftar submenu yang ada.
    Kemungkinan Help And Manual tidak mengenal WinHelp Compiler di kompiter Anda. Jika demikian, masuklah ke Compiler Options lewat menu Tools>Customize, klik tab Compilers. Aturlah path tempat WinHelp Compiler berada

    Jika sudah, klik OK.
  • Kembali ke proses kompilasi, kita sekarang bisa mengatur di mana akan meletakkan output kompilasi file help kita. Sebaiknya kita letakkan saja dalam satu folder dengan aplikasi kita

    Klik OK.
  • Kompilasi berjalan. Kita tunggu sampai selesai. Window Compiler Messages akan ditampilkan dengan data kompilasi file help kita, seperti berikut
  • Oke, kompilasi berjalan lancar. Berikut tampilan file help (.HLP) kita

Pembuatan file help dengan Help And Manual telah selesai. File help ini siap dipakai dalam aplikasi kita nanti.

Pembuatan Aplikasi [Delphi]

Sekarang kita buat aplikasi kita. Jalankan Delphi, dan buat sebuat aplikasi baru.

  • Tambahkan sebuah button ke form, hingga tampilan form kira-kira seperti ini
  • Sekarang beralih ke Project Options lewat menu Project>Options. Sesuaikan nama help file ke file help yang telah kita buat sebelumnya

    Klik OK.
  • Kembali ke form, klik pada Button1, dan lewat Object Inspector, atur properti HelpType ke htContext dan HelpContext dengan nilai 1001 - nilai yang kita masukkan pada langkah 4, proses pembuatan file help dengan Help And Manual.
  • Selesai. Simpan project kita dan tekan F9 untuk menjalankan aplikasi. Setelah muncul window aplikasi, letakkan pointer mouse ke atas Button1 pada form, dan tekan F1 pada keyboard.
