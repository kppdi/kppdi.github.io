TDBNavigatorPlus & TDBNavigatorPlusX

Grup KPPDI
Ini dua komponen alternatif bagi TDBNagivator milik Delphi. Komponen ini awalnya hanya TDBNavigator yang dibuat oleh William R. Florac, dengan menambahkan property Captions agar user dapat menampilkan tulisan pada tiap tombol navigator ini.

Berangkat dari ide menarik ini, saya membuat duplikasi komponen ini, mengubah Captions menjadi berbahasa Indonesia secara default dan menambahkan property ImageCancel, ImageFirst, dst., sesuai tiap button pada navigator. Image-image ini  secara default terlihat pada screenshot di atas (navigator paling bawah). Untuk navigator yang telah saya ubah glyphnya adalah yang paling atas. Dan yang asli buatan William R. Florac adalah yang di tengah.

INSTALASI

Untuk menginstal komponen ini, silahkan download dan ekstrak komponen dari seksi download di akhir tulisan ini.

  1. Ekstrak ke sebuah folder. Tambahkan folder tersebut ke Library Path milik Delphi [ Tools>Environment Options>Library Path ]
  2. Untuk Delphi 6 dan Delphi 7, pilih menu Component>Install Component, dan pilih DBNavPlus.pas. Klik Compile, klik Install. Ulangi langkah ini untuk file kedua, uNavPlusGlyph.pas.
  3. Untuk BDS2006, klik menu  File>Open dan pilih file DBNavPlusX_D2006.dpk. Klik Compile dan Klik Install.
  4. Anda sudah memiliki kedua komponen terinstall di Delphi Anda, pada page Fitco di Component Palette

Pada screenshoot di atas Anda dapat melihat kolaborasi TDBNavigatorPlus & TDBNavigatorPlusX dengan ZeosDBO - MySQL :)
