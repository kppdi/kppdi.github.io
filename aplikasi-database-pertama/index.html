Aplikasi Database Pertama

Grup KPPDI

Ehem.... Sebenarnya blog ini awalnya bukan untuk bahas database, tapi beberapa pengunjung me-request database, jadi saya posting juga.
Saya akan mulai dari aplikasi pertama: Ikan Explorer.
Software: Delphi 3, Database Explorer, BDE (Borland Database Engine), Demo database bawaan Delphi 3, diuji pada Windows XP.


Pembuatan Alias Database

Lansung saja ke langkah-langkah pembuatannya:

  1. Jalankan Delphi 3 (Delphi 4, 5, 6 dan 7 seharusnya tidak masalah). Klik menu Database>Explore...
  2. Pada window Database Explorer, klik kanan pada Tab Databases, Item Databases, klik New...

  3. Pilih Standard pada combobox Database Driver Name.
    Standard adalah pilihan tipe database default yang merupakan Driver Database Paradox. Klik OK.
  4. Pada nama database, masukkan Nama Alias untuk database Anda.
    Alias berguna jika path database Anda cukup panjang dan sulit diingat. Dengan Alias, tinggal Alias saja yang disebutkan, dan driver database akan mengacu ke path yang terkait dengan Alias ini. Tekan Enter.
  5. Selanjutnya, klik tombol elipsis [...] di bagian kanan baris yang bertuliskan PATH... Pilih Path DBDemos, yaitu %Delphi3Folder%\Demos\DATA.
    Path ini nantinya akan jadi acuan jika aplikasi mengakses database dengan Alias yang Anda tentukan pada langkah 4.
  6. Sudah itu, klik kanan lagi pada nama Database/Alias di bagian kiri, klik Apply. Pada dialog konfirmasi yang muncul, klik OK.
  7. Tutup Database Explorer



Pembuatan Aplikasi
Kini kita buat aplikasinya di Delphi:

  1. Buat aplikasi baru. Tambahkan sebuah sebuah TTable & TDataSource dari page Data Access.
  2. Tambahkan juga TDBGrid, TDBNvigator, TDBImage dari page Data Controls. Juga sebuah TButton dari page Standard dan sebuah TOpenPictureDialog dari page Dialogs.
  3. Atur properti Database milik Table1 pada form sesuai nama Alias yang Anda tentukan pada Pembuatan Alias Database di atas.
    Atur juga properti TableName, isikan BIOLIFE.DB.

    Set properti Active ke True.
  4. Untuk DataSource1, set properti Dataset ke Table1.
  5. Set properti DataSource milik DBGrid1 dan DBNavigator1 ke DataSource1.
  6. Set juga properti DataSource milik DBImage1 ke DataSource1, dan DataFiled ke GRAPHIC.

    Sampai di sini, pada form sudah nampak data-data yang tersimpan dalam tabel BIOLIFE.DB.
  7. Sekarang kita akan membuat kode untuk mengubah gambar record yang aktif (saat proses edit atau input data baru).
    Klik ganda Button1 dan masukkan kodenya:

    [sourcecode language='delphi']
    procedure TForm1.Button1Click(Sender: TObject);
    begin
    if openpicturedialog1.Execute then
    DBImage1.Picture.LoadFromFile(OpenPictureDialog1.fileName);
    end;
    [/sourcecode]

    Beri Caption untuk button ini dengan "Ubah Gambar...".

  8. Jalankan aplikasi Anda
